예외에는 일반 예외랑 실행 예외가 있다
일반예외는 코드작성시 뜨는 오류로 자동으로 찾아줌
그러나 실행 예외는 컴파일시 생기는 오류로 이는 사전에 예방을 해주던가 사용자가 따로 코드를 통해 처리를 해주어야 한다.

이런 오류를 처리하는것을 예외 처리 라고 하고
try - catch - finally 라는 예외처리 코드를 작성함
각각은 블록으로 구성되어 있는데 
try 는 예외 발생 가능 코드가 위치
catch는 예외 발생이 없다면 작동하지 않고 finally로 넘어간다 만약 예외가 발생하게 된다면
catch 블록 으로 이동하여 예외 처리 코드를 실행합니다.
Fimally는 생략이 가능하지만 예외 발새 여부와 상관없이 사용할 것이 있을때 사용
